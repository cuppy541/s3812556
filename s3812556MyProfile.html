Project idea

My idea for an I.T. project is an extension to the current application and addon service twitch. I would like to see an in-game overlay system where I can look for addon or mods. install them, uninstall them and with just a reload of the UI have them up and running in seconds. You could also use it for writing down some notes if you are a content creator for the site and have an idea mid game for a mod or addon and you don’t want to forget it.

I can’t say that I can give much statistics on this idea because I haven’t really seen or heard anyone complain about the problem. But there is nothing worse than having to exit a game completely to add a mod or add-on. Most add-ons are just UI changes so if you just reload the UI in world of warcraft for example you should be able to see the new add-on you just installed. Just like a smartphone, installing applications does not require to restart the phone. So just a little add-on to a game should not require you to close the game completely.

Above was a Picture on the current steam overlay and a few features it has. Now twitch already has an overlay system but it is only for streamers to have customs things over their stream for viewer to see. So, I would like to have one for add-ons and mods, that we can use while playing a game and not switching out windows to go to it. Features I would like to see would be;

  • Clean easy to use overlay UI.
  • Add-on/mods finder.
  • Social tab to talk to friends not in game.
  • Forum tab to see what people are saying about the game you are in.
  • Creators pages, to submit bugs or to provide feedback.
    • Seeing some of the features I would like to see in the overlay, I will now go into each topic in more detail. I would like the UI to be clean and easy to use. It would need to be slightly transparent as to still see what is going on in-game. At the top you would have tabs going from left to right displaying the different categories such as home, add-ons, social, forums, feedback and help. The home tab would just have an overview of all recent activity. The colour theme would still be the twitch purple but a little transparent to make it look more of a lavender colour. The add-ons tab would have your usual search, browse, top add-ons trending and paid if they wanted to go down that route. If you are using this overlay, you would most likely know what add-ons your looking for so having the search button first would just be easier. The social tab would have your friends online and who is offline, you will be able to click onto your friend’s name and talk to them via private message. You will also be able to view your friend’s profile as well. The forum tab will be simple with a search button and a create button, it would also have a trending and new part of the page with about the 10 of each on the page. The last tab would be where you search and see creator’s pages that have all their add-ons so you can provide feedback or even submit bugs/problems with the addons they are providing. You will be able to activate the overlay via a keystroke of your choosing.

    You will not need much hardware apart from a keyboard to put the keystroke into to activate the overlay and maybe a few computers to put input the code. My idea is just about adding to already built software. Twitch and if they still use curse’s software all they would need to do is open their source code and add the code for the add-ons overlay.

    The core software is already built being twitch. The software that needs to be made is more of an extension. From what I can recall I have not been a part of a project before, but from what I can tell some of the roles and skill required for my project would include a project manager, a chief technology officer, 1-3 senior software developers, 1-5 junior software developers, some testers and a maintenance crew for ongoing support to this software. In twitch’s pool of staff members, it would be very feasible to get the job done and make this come to life.

    If my project idea was successful, I believe it would be a very beneficial addiction to an already growing company and its users. Being able to add add-ons on the fly, talk to content creators and provide feed and also search forums all without alt tabbing out of game. I believe it if was to be created it would have a welcoming reception to the users of Twitch and Curse. The impact it would have on me would be high one. I am always installing and uninstall add-ons and mods to suite my gameplay at the time. So being able to do that with a smart overlay I think would improve my gameplay slightly.
